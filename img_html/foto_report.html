Национальная аллея.
На аллее сосредоточены все основные памятники и монументы американской столицы.
С воздуха аллея представляет собой крест: на севере ( слева ) - Белый дом,
на юге ( справа ) - мемориал Джефферсона, на востоке ( сверху ) - Капитолий,
на западе ( снизу ) - мемориал Линкольна, в центре - монумент Вашингтона.
А между этими архитектурными колоссами располагаются парки и Ботанический сад.
